# BCN Digital Local Landing

Landing page de sales enablement para agencia de digitalizacion de negocios locales en Barcelona.

## Stack

- Next.js (App Router)
- Tailwind CSS
- Framer Motion

## Desarrollo

1. Instala dependencias:

```bash
npm install
```

2. Ejecuta en local:

```bash
npm run dev
```

3. Abre:

`http://localhost:3000`
